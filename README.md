React/Next.js + Zustand + TypeScript

## Getting Started

Доступы для входа, как в документе к тестовому.

First, run the development server:

```bash
npm i
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
